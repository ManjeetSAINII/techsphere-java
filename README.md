# TechSphere Java E-Commerce App

## 🚀 Live Demo

https://techsphere-java-production.up.railway.app

## 🛠 Tech Stack

* Java (JSP/Servlet)
* MySQL (Railway)
* Apache Tomcat
* Railway (Deployment)

## ✨ Features

* Product listing
* User authentication
* Cart system
* Order management

## ⚙️ Setup

1. Clone repo
2. Add environment variables:

   * DB_CONNECTION_STRING
   * DB_USERNAME
   * DB_PASSWORD
3. Run project

## 🌐 Deployment

Deployed on Railway with MySQL integration
