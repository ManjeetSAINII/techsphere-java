# TechSphere Java E-Commerce App

## 🚀 Live Demo
https://techsphere-java.onrender.com

## 🛠 Tech Stack

* Java (JSP/Servlet)
* MySQL (TiDB)
* Apache Tomcat
* Render (Deployment)

## ✨ Features

* Product listing
* User authentication
* Cart system
* Order management

## ⚙️ Setup

1. Clone repo
2. Add environment variables:

   * DB_CONNECTION_STRING
   * DB_USERNAME
   * DB_PASSWORD
3. Run project

## 🌐 Deployment

Deployed on Render with MySQL integration(TiDB)
